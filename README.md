# MobliePlayer


> 一个基于MobileVLCKit的播放器,正在不断优化

###### 下载完成后请到该文件夹下执行

```
pod install
```

###### MobileVLCKit框架会比较大，pod install时间会较长，大约有800M

> 引用的框架如下

```

pod 'MobileVLCKit'
pod 'Masonry'
pod 'CocoaHTTPServer'
```

###### 实现功能如下

```
1. 音量、亮度调整
2. 视频进度调整
3. 全屏、及其屏幕强制旋转
4. 播放各种格式的视频
5. 网络视频播放
6. 文件/文件夹查看
7. 电脑传输到手机
```
